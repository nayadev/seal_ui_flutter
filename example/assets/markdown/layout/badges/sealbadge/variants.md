# SealBadge — Variants

A compact label badge built on `ShadBadge`. Seven semantic variants
cover brand, status, and state use cases.

## Usage

```dart
SealBadge.primary(child: const Text('New'))
SealBadge.accent(child: const Text('Beta'))
SealBadge.secondary(child: const Text('Draft'))
SealBadge.outline(child: const Text('v1.0'))
SealBadge.success(child: const Text('Active'))
SealBadge.warning(child: const Text('Pending'))
SealBadge.error(child: const Text('Expired'))
```

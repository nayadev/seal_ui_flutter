# SealAlert — Success

An inline **success** alert with a colored border and icon. Use to confirm that an action completed successfully.

## Without title

```dart
SealAlert.success(
  description: 'Your changes have been saved successfully.',
);
```

## With title

```dart
SealAlert.success(
  title: 'All done!',
  description: 'Your changes have been saved successfully.',
);
```

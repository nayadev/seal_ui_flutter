# SealAlert — Info

An inline **informational** alert with a colored border and icon.

## Without title

```dart
SealAlert.info(
  description: 'You can add components to your app using the CLI.',
);
```

## With title

```dart
SealAlert.info(
  title: 'Heads up!',
  description: 'You can add components to your app using the CLI.',
);
```

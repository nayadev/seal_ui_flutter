# SealFilledButton — With Icon

Any `SealFilledButton` variant accepts an optional leading **icon** via the `icon` parameter.

## Usage

```dart
SealFilledButton.gradient(
  label: 'Explore',
  icon: Icons.rocket_launch_rounded,
  onPressed: () {},
);
```

## Other variants with icon

```dart
// Primary with icon
SealFilledButton.primary(
  label: 'Save',
  icon: Icons.save_rounded,
  onPressed: () {},
);

// Accent with icon
SealFilledButton.accent(
  label: 'Share',
  icon: Icons.share_rounded,
  onPressed: () {},
);
```

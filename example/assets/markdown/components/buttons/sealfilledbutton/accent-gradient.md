# SealFilledButton — Accent Gradient

A filled button with an **accent gradient** background. Use to draw maximum visual attention to a feature-highlighting action.

## Usage

```dart
SealFilledButton.accentGradient(
  label: 'Boost',
  onPressed: () {},
);
```

## Loading state

```dart
SealFilledButton.accentGradient(
  label: 'Boosting…',
  isLoading: true,
  onPressed: null,
);
```

# SealFilledIconButton — Primary

A filled icon-only button using the **primary** color. Use for compact actions with a clear icon affordance and a tooltip for accessibility.

## Usage

```dart
SealFilledIconButton.primary(
  icon: Icons.add_rounded,
  tooltip: 'Add item',
  onPressed: () {},
);
```

## Disabled state

```dart
SealFilledIconButton.primary(
  icon: Icons.add_rounded,
  tooltip: 'Add item',
  onPressed: null,
);
```

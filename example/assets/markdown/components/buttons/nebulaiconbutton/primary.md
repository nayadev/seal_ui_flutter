# NebulaIconButton — Primary

A transparent icon button with a **primary** color tint. Use for subtle icon actions in toolbars, app bars, or inline controls.

## Usage

```dart
NebulaIconButton.primary(
  icon: Icons.close_rounded,
  tooltip: 'Close',
  onPressed: () {},
);
```

## Disabled state

```dart
NebulaIconButton.primary(
  icon: Icons.close_rounded,
  tooltip: 'Close',
  onPressed: null,
);
```

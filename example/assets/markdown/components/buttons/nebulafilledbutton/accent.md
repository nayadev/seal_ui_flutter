# NebulaFilledButton — Accent

A filled button using the **accent** color palette. Ideal for positive confirmation actions and secondary CTAs.

## Usage

```dart
NebulaFilledButton.accent(
  label: 'Continue',
  onPressed: () {},
);
```

## Loading state

```dart
NebulaFilledButton.accent(
  label: 'Processing…',
  isLoading: true,
  onPressed: null,
);
```

## Disabled state

```dart
NebulaFilledButton.accent(
  label: 'Continue',
  onPressed: null,
);
```

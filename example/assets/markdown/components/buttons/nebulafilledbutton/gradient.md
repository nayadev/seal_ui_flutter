# NebulaFilledButton — Gradient

A filled button with a **gradient** background derived from the primary color tokens. Use for hero or spotlight actions.

## Usage

```dart
NebulaFilledButton.gradient(
  label: 'Launch',
  onPressed: () {},
);
```

## Loading state

```dart
NebulaFilledButton.gradient(
  label: 'Launching…',
  isLoading: true,
  onPressed: null,
);
```

## Disabled state

```dart
NebulaFilledButton.gradient(
  label: 'Launch',
  onPressed: null,
);
```

# NebulaFilledButton — Accent Secondary

A filled button using the **accent secondary** color. Use for supporting actions that complement the primary flow.

## Usage

```dart
NebulaFilledButton.accentSecondary(
  label: 'Confirm',
  onPressed: () {},
);
```

## Loading state

```dart
NebulaFilledButton.accentSecondary(
  label: 'Confirming…',
  isLoading: true,
  onPressed: null,
);
```

## Disabled state

```dart
NebulaFilledButton.accentSecondary(
  label: 'Confirm',
  onPressed: null,
);
```

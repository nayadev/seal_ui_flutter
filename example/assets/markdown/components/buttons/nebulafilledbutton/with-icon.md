# NebulaFilledButton — With Icon

Any `NebulaFilledButton` variant accepts an optional leading **icon** via the `icon` parameter.

## Usage

```dart
NebulaFilledButton.gradient(
  label: 'Explore',
  icon: Icons.rocket_launch_rounded,
  onPressed: () {},
);
```

## Other variants with icon

```dart
// Primary with icon
NebulaFilledButton.primary(
  label: 'Save',
  icon: Icons.save_rounded,
  onPressed: () {},
);

// Accent with icon
NebulaFilledButton.accent(
  label: 'Share',
  icon: Icons.share_rounded,
  onPressed: () {},
);
```

# NebulaTextButton — With Icon

`NebulaTextButton` supports an optional **trailing icon** via the `icon` parameter.

## Usage

```dart
NebulaTextButton.primary(
  label: 'Skip',
  icon: Icons.arrow_forward_rounded,
  onPressed: () {},
);
```

## Other variants with icon

```dart
NebulaTextButton.gradient(
  label: 'See all',
  icon: Icons.chevron_right_rounded,
  onPressed: () {},
);
```

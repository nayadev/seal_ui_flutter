# SealTextButton — Primary

A text-only button with **primary** color. Use for low-emphasis actions like "Learn more" or inline navigation.

## Usage

```dart
SealTextButton.primary(
  label: 'Learn more',
  onPressed: () {},
);
```

## Loading state

```dart
SealTextButton.primary(
  label: 'Loading…',
  isLoading: true,
  onPressed: null,
);
```

## Disabled state

```dart
SealTextButton.primary(
  label: 'Learn more',
  onPressed: null,
);
```

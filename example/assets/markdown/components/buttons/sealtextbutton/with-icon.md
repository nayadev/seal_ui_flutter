# SealTextButton — With Icon

`SealTextButton` supports an optional **trailing icon** via the `icon` parameter.

## Usage

```dart
SealTextButton.primary(
  label: 'Skip',
  icon: Icons.arrow_forward_rounded,
  onPressed: () {},
);
```

## Other variants with icon

```dart
SealTextButton.gradient(
  label: 'See all',
  icon: Icons.chevron_right_rounded,
  onPressed: () {},
);
```

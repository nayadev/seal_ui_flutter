# SealOutlineIconButton — Primary

An outlined icon-only button with a **primary** color stroke. Use when a ghosted style is preferred for icon actions.

## Usage

```dart
SealOutlineIconButton.primary(
  icon: Icons.share_rounded,
  tooltip: 'Share',
  onPressed: () {},
);
```

## Disabled state

```dart
SealOutlineIconButton.primary(
  icon: Icons.share_rounded,
  tooltip: 'Share',
  onPressed: null,
);
```

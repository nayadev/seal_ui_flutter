# NebulaOutlineButton — With Icon

Any `NebulaOutlineButton` variant supports an optional **leading icon** via the `icon` parameter.

## Usage

```dart
NebulaOutlineButton.primary(
  label: 'Favorite',
  icon: Icons.star_outline_rounded,
  onPressed: () {},
);
```

## Other variants with icon

```dart
NebulaOutlineButton.accent(
  label: 'Bookmark',
  icon: Icons.bookmark_outline_rounded,
  onPressed: () {},
);

NebulaOutlineButton.gradient(
  label: 'Share',
  icon: Icons.share_rounded,
  onPressed: () {},
);
```

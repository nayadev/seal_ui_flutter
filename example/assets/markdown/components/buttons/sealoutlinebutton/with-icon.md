# SealOutlineButton — With Icon

Any `SealOutlineButton` variant supports an optional **leading icon** via the `icon` parameter.

## Usage

```dart
SealOutlineButton.primary(
  label: 'Favorite',
  icon: Icons.star_outline_rounded,
  onPressed: () {},
);
```

## Other variants with icon

```dart
SealOutlineButton.accent(
  label: 'Bookmark',
  icon: Icons.bookmark_outline_rounded,
  onPressed: () {},
);

SealOutlineButton.gradient(
  label: 'Share',
  icon: Icons.share_rounded,
  onPressed: () {},
);
```

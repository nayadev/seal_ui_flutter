# SealOutlineButton — Gradient

An outlined button with a **gradient** text/stroke style. Use for stand-out secondary actions.

## Usage

```dart
SealOutlineButton.gradient(
  label: 'Explore',
  onPressed: () {},
);
```

## Loading state

```dart
SealOutlineButton.gradient(
  label: 'Exploring…',
  isLoading: true,
  onPressed: null,
);
```

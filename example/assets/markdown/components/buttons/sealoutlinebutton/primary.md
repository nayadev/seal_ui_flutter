# SealOutlineButton — Primary

An outlined button with a **primary** color stroke. Use as a secondary action alongside a filled primary button.

## Usage

```dart
SealOutlineButton.primary(
  label: 'Cancel',
  onPressed: () {},
);
```

## Loading state

```dart
SealOutlineButton.primary(
  label: 'Cancelling…',
  isLoading: true,
  onPressed: null,
);
```

## Disabled state

```dart
SealOutlineButton.primary(
  label: 'Cancel',
  onPressed: null,
);
```

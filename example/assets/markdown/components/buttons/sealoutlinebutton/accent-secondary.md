# SealOutlineButton — Accent Secondary

An outlined button with an **accent secondary** color stroke.

## Usage

```dart
SealOutlineButton.accentSecondary(
  label: 'Info',
  onPressed: () {},
);
```

## Loading state

```dart
SealOutlineButton.accentSecondary(
  label: 'Loading…',
  isLoading: true,
  onPressed: null,
);
```

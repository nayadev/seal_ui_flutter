# SealOutlineButton — Accent

An outlined button with an **accent** color stroke. Use for secondary accent-themed actions.

## Usage

```dart
SealOutlineButton.accent(
  label: 'Details',
  onPressed: () {},
);
```

## Loading state

```dart
SealOutlineButton.accent(
  label: 'Loading…',
  isLoading: true,
  onPressed: null,
);
```
